• Flexible Box Layout Module
  • Everything is set to be in "flex flow"
  • You need a container (display: flex)
  • Child elements are flex items
  • You can now set on the container, flex-direction, flex-wrap, flex-flow, justify-content, align-tiems
  • Flexbox has a main (horizontal) and cross (vertical) axis
  • These can be reversed
  • flex-direction: row (main), columen (cross)
  • Flex-direction sets the main access
  • The left is called the main start, and right is main end (changing direction reverses these
  • Default is for everything to be on one line (flex-wrap controls this)
  • Flex-flow is shorthand for direction and flow (i.e. flex-flow: row nowrap)
  • Justify-content aligns item along the main axis
  • It has space-between (like justify, but last item will align to main end
  • space-around gives extra spacing
  • Align-itmes aligns via the cross access
  • Align-content similar to justify-content
  • Flex items have flex-grow, flex shrink, flex-basis, flex, align-self and order properties.
  • Flex-grow sets the width the element takes up relative to the container (standard width will override)
  • Flex-shrink is similar but kicks in when the window decreases in size
  • Both take a whole number or decimal (0.25 for a quarter of the space, etc.)
  • Flex-basis is like a width property and can take any css numerical value
  • Flex is the shorthand for the previous 3 (grow, shrink, basis)
  • Align-self allows to align a specific element different than it siblings
  • Order takes a positive or negative whole number and allows you reorder a specific item
